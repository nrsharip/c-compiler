		C compiler using:

1) Launch command line
2) The source code file has to have “.c” extension
3) Write “cc.exe”
4) Write after the space, the source code file path. 
5) There is also an option ability to specify an absolute offset of program in address space

PLEASE, NOTE: 

If the file name is not specified, the input 
line “test.c 7C00” will be accepted as default
